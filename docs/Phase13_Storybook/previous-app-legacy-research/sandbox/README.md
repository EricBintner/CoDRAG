# Visual Design Sandbox

Quick isolated environments to compare visual design directions.

## Variations

| Variation | Theme | Description |
|-----------|-------|-------------|
| **A** | Refined Dark | Polish current style with better tokens |
| **B** | Soft Neutral | Lighter, warmer, softer aesthetic |
| **C** | Glassmorphism | Modern glass effects, depth |
| **D** | Bold Accent | High contrast, vibrant accents |

## Running

Each variation is a standalone HTML file. Open directly in browser:

```bash
# From this directory
open variation-a-refined-dark/index.html
open variation-b-soft-neutral/index.html
open variation-c-glass/index.html
open variation-d-bold/index.html
```

Or use live server for hot reload:

```bash
npx serve .
```

## Components Showcased

Each variation renders the same components:
- Button (primary, secondary, ghost, destructive)
- Input fields
- Cards
- Dialog/Modal
- Tabs
- Chat bubble
- Avatar
- Badge/Tag

## Quick Comparison

Open all four in browser tabs and switch between them to compare.
