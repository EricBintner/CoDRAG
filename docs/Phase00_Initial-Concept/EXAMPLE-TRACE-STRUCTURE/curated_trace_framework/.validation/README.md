# Validation Directory

This directory contains machine-readable validation reports generated by Pass 3.

## Files

- `orphans.json` — entries with no plan/purpose links
- `broken_links.json` — links pointing to non-existent entries
- `stale_entries.json` — entries not updated recently

## Schemas

### orphans.json
```json
{
  "generated": "ISO-8601 date",
  "total_orphans": 0,
  "entries": []
}
```

### broken_links.json
```json
{
  "generated": "ISO-8601 date",
  "total_broken": 0,
  "links": []
}
```

### stale_entries.json
```json
{
  "generated": "ISO-8601 date",
  "total_stale": 0,
  "entries": []
}
```
